Правила посещения

  1. Регистрация на сайте обязательна для участия в виртуальных турах
  2. Рекомендуется стабильное интернет-соединение (от 10 Мбит/с)
  3. Для VR-туров необходимы совместимые устройства
  4. Запрещена коммерческая съемка без разрешения
  5. Уважайте других посетителей во время групповых туров

Как добраться

Наш виртуальный музей доступен из любой точки мира. Для офлайн-мероприятий:

  • Адрес: г. Москва, ул. Музейная, д. 1
  • Метро: Станция "Культурная" (выход к музеям)
  • Автобусы: № 144, 256, 333 (остановка "Художественный музей")
